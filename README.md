## Building

```shell
$ go get github.com/constabulary/gb/...
$ git clone git@github.com/atongen/gosaic.git
$ cd gosaic
$ gb build all
```

### Requirements

* golang 1.6.x or later
* [gb](https://getgb.io/)
* sqlite3
